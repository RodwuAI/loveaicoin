### 📏 设计硬性规范（从教训中提炼）
| 规范项 | 标准 | 来源 |
|--------|------|------|
| 标题字体 | H1=48px, H2=36px, H3=24px, H4=20px | 官网项目 |
| 正文字体 | Body=16px, Small=14px, Caption=12px | 官网项目 |
| 行高 | 标题1.2, 正文1.6 | 官网项目 |
| 字重 | 标题700, 正文400, 强调600 | 官网项目 |
| 间距基准 | 8px网格（8/16/24/32/48/64/96） | 设计系统 |
| 卡片圆角 | 16px | Phantom参考 |
| 按钮圆角 | 12px | Phantom参考 |
| 容器最大宽 | 1200px，水平居中 | 官网项目 |
| 响应式断点 | Mobile<768px, Tablet<1024px, Desktop≥1024px | 通用 |
| 视觉偏差容差 | ≤5px，超过即不通过 | 吴总指令 |
| 品牌色 | 藏蓝#1B2D6B / 金色#C5975B / 浅金#D4B88C | Logo |
| 字体 | Inter（英文）+ PingFang SC（中文） | 品牌规范 |

### ✅ 最佳实践
- 设计前先截图参照物同区域，再动手
- 每个页面交付前用computed style验证关键数值
- 颜色克制：白底为主，金色只做点缀，不大面积铺色
- 大量留白，宁可空也不要挤
