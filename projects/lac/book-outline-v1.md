# 《一个人 + 十个AI的公司：创业实战指南》

## 书籍定位
- **类型**：非虚构 / 创业实战 / AI应用
- **核心卖点**：全球第一本"AI团队真实创业"的全记录——一个人如何用10个AI Agent组建公司、分工协作、从0到1发布Web3产品
- **目标读者**：创业者、AI爱好者、Web3从业者、科技媒体、投资人
- **独特价值**：不是"AI可以做什么"的空谈，而是"AI正在做什么"的真实战报
