## 3. Learn-to-Earn（学习挖矿）详细规则

### 3.1 奖励行为表

| 行为 | 基础奖励(LAC) | 条件 | 冷却时间 | 日上限(LAC) |
|------|-------------|------|---------|-----------|
| 完成一节课程（L1入门） | 50 | 通过测验≥60分 | 无（每课仅首次全额） | 500 |
| 完成一节课程（L2进阶） | 100 | 通过测验≥60分 | 无 | 500 |
| 完成一节课程（L3高级） | 200 | 通过测验≥60分 | 无 | 500 |
| 完成一节课程（L4专家） | 400 | 通过测验≥60分 | 无 | 500 |
| 通过AI测验（随课程附带） | 含在课程奖励中 | 分数影响Quality_Score | — | — |
| 获得认证证书（完成整个课程） | 500 | 完成该课程所有章节 | 每课程仅1次 | 不计入日上限 |
| 每日AI签到 | 30 | 回答1道AI知识题，正确率≥50% | 每日1次 | 30 |

### 3.2 连续签到/学习加成倍率

| 连续天数 | 倍率 | 常量名 |
|---------|------|--------|
| 1-2天 | 1.0x | `STREAK_BASE` |
| 3天 | 1.2x | `STREAK_3D` |
| 7天 | 1.5x | `STREAK_7D` |
| 14天 | 2.0x | `STREAK_14D` |
| 30天 | 3.0x | `STREAK_30D` |
| 60天 | 5.0x | `STREAK_60D` |
| 90天+ | 6.0x | `STREAK_90D` |

**规则：**
- 连续加成**可突破日上限**，最高至基础日上限的 **3倍**（即Learn最高1,500 LAC/天）
- 每月1次免费补签，额外补签消耗50 LAC
- 断签后倍率归1.0x，连续天数归0

### 3.3 课程难度系数

| 级别 | 系数 | 常量名 | 课程时长 |
|------|------|--------|---------|
| L1 入门 | 1.0 | `DIFF_L1` | 5-10分钟 |
| L2 进阶 | 1.5 | `DIFF_L2` | 15-30分钟 |
| L3 高级 | 2.5 | `DIFF_L3` | 30-60分钟 |
| L4 专家 | 4.0 | `DIFF_L4` | 2-5小时 |

### 3.4 测验质量系数（Quality_Score）

| 测验分数 | Quality_Score | 说明 |
|---------|--------------|------|
| 0-59分 | 0.1 | 不及格，惩罚性极低奖励 |
| 60分 | 0.3 | 及格线 |
| 70分 | 0.475 | |
| 80分 | 0.65 | |
| 90分 | 0.825 | |
| 100分 | 1.0 | 满分 |

**公式：**
```
Quality_Score(x) =
  0.1                          if x < 60
  0.3 + 0.7 × (x - 60) / 40   if 60 ≤ x ≤ 100
```

### 3.5 完整奖励计算公式

```
Learn_Reward = Base_Rate × Difficulty × Quality_Score × Streak_Multiplier × Level_Bonus × Scarcity_Factor
```

| 参数 | 来源 | 取值范围 |
|------|------|---------|
| `Base_Rate` | 3.1节表 | 50/100/200/400 LAC |
| `Difficulty` | 3.3节表 | 1.0 / 1.5 / 2.5 / 4.0 |
| `Quality_Score` | 3.4节公式 | 0.1 ~ 1.0 |
| `Streak_Multiplier` | 3.2节表 | 1.0 ~ 6.0 |
| `Level_Bonus` | 7.2节 | 1.00 ~ 1.25 |
| `Scarcity_Factor` | 当日Learn池剩余/日总额 | 0.0 ~ 1.0 |

**日上限规则：**
```
Base_Daily_Cap = 500 LAC
Effective_Daily_Cap = min(Base_Daily_Cap × 3, Base_Daily_Cap × Streak_Factor_Cap)

其中 Streak_Factor_Cap:
  streak < 14天:  cap = 1.0（不突破）
  streak 14-29天: cap = 1.5
  streak 30-59天: cap = 2.0
  streak 60天+:   cap = 3.0
```

## 5. Teach-to-Earn（教导挖矿）详细规则

### 5.1 奖励行为表

| 行为 | 基础奖励(LAC) | 条件 | 冷却 | 日上限(LAC) |
|------|-------------|------|------|-----------|
| 发布教学内容（文章/视频） | 100（基础） | Lv.3+，AI审核通过 | 每日最多5篇 | 2,000 |
| 内容获得互动（T+7结算） | 按公式计算 | 7天审核期后结算 | — | 2,000 |
| 回答社区问题（被采纳） | 15 | 被提问者标记为最佳答案 | 无 | 含在2,000总上限 |
| 内容获赞 | 1 LAC/赞 | 去重（每人每篇仅1赞） | — | — |
| 内容被收藏 | 5 LAC/收藏 | 去重 | — | — |
| 内容被评论 | 3 LAC/评论 | 去重（每人每篇仅1条计入） | — | — |

### 5.2 T+7结算公式

```
Teach_Reward = Interaction_Score × Quality_Multiplier × Originality_Score

Interaction_Score = likes × 1 + comments × 3 + bookmarks × 5

Quality_Multiplier = AI_Score × 0.6 + Community_Score × 0.4
  - AI_Score: 0.0 ~ 3.0（GPT-4级模型评估）
  - Community_Score: 0.0 ~ 3.0（用户评分中位数 × 0.6）

Originality_Score:
  AI查重 < 30%    → 1.0（高原创）
  AI查重 30%-60%   → 0.6（中等原创）
  AI查重 60%-80%   → 0.2（低原创）
  AI查重 > 80%     → 0.0（抄袭，归零）
```

### 5.3 发放规则

- 结算周期：**T+7**（内容发布7天后结算）
- 日上限：**2,000 LAC**（含T+7结算到账和实时回答奖励）
- 每日最多**5篇**内容获得奖励
- 结算后自动发放至用户钱包
