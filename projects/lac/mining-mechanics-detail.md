# LAC 三维挖矿机制详细定义

> CEO指令（2026-02-20）：所有发生在LAC网站上的、跟AI传播/使用/拥抱相关的行为，都给空投奖励。

| 行为 | 奖励触发 | 备注 |

| 观看AI短视频 | 按观看时长计 | 需防刷机制 |

| 阅读AI文章/资料 | 阅读时长+互动 | |
