## 一、场景概述

### 1.1 核心概念

营销内容工厂是**标准化的批量视频生产系统**：
- 建立标准化提示词模板库
- 批量生成产品宣传视频
- 实现营销内容工业化生产

### 1.2 应用场景矩阵

| 场景类型 | 应用示例 | 频次 | 目标受众 |
|---------|---------|------|---------|
| **平台宣传** | 新品发布、功能更新 | 月度 | 全量用户 |
| **名人营销** | 入驻官宣、活动预告 | 每周 | 粉丝群体 |
| **用户故事** | 成功案例、UGC二创 | 每日 | 潜在用户 |
| **节日营销** | 春节/情人节/双11 | 节日 | 全量用户 |
| **日常运营** | 日签、精选、热点 | 每日 | 全量用户 |

### 1.3 商业价值

| 价值点 | 量化指标 |
|-------|---------|
| **内容产能** | 日均100+条，满足全渠道需求 |
| **成本优势** | 单条成本¥1.55 vs 外包¥800+ |
| **快速迭代** | A/B测试多种创意，数据驱动 |
| **品牌一致** | 统一模板确保调性统一 |

class VideoCropper:
    """多平台视频裁剪器"""
    
    CROP_CONFIGS = {
        "douyin": {
            "aspect_ratio": "9:16",
            "resolution": (1080, 1920),
            "safe_zone": {"top": 0.1, "bottom": 0.1, "left": 0, "right": 0},
            "hooks": {"first_3s": True}  # 前3秒强吸引点
        },
        "xiaohongshu": {
            "aspect_ratio": "3:4",
            "resolution": (1080, 1440),
            "safe_zone": {"top": 0.05, "bottom": 0.05, "left": 0.05, "right": 0.05},
            "style": "polished"  # 精致风格
        },
        "bilibili": {
            "aspect_ratio": "16:9",
            "resolution": (1920, 1080),
            "safe_zone": {"top": 0.05, "bottom": 0.05, "left": 0.05, "right": 0.05},
            "subtitle": True  # 添加字幕
        }
    }
    
    async def crop_for_platform(self, video_path: str, platform: str) -> str:
        """为指定平台裁剪视频"""
        config = self.CROP_CONFIGS[platform]
        
        # 使用FFmpeg进行智能裁剪
        cmd = f"""
        ffmpeg -i {video_path} 
        -vf "crop={self.calculate_crop(config)}" 
        -c:v libx264 -crf 23 
        {platform}_output.mp4
        """
        
        await self.execute(cmd)
        return f"{platform}_output.mp4"

### 6.2 批量生成引擎

```python
class BatchVideoEngine:
    """批量视频生成引擎"""
    
    def __init__(self):
        self.queue = RedisQueue()
        self.workers = 10
        self.concurrency = 5  # Seedance并发限制
    
    async def submit_batch(self, requests: List[VideoRequest]) -> BatchJob:
        """提交批量生成任务"""
        job_id = generate_uuid()
        
        # 拆分为子任务
        for idx, req in enumerate(requests):
            task = GenerationTask(
                job_id=job_id,
                task_id=f"{job_id}_{idx}",
                request=req,
                priority=self.calculate_priority(req),
                status="pending"
            )
            await self.queue.enqueue(task)
        
        return BatchJob(id=job_id, total=len(requests))
    
    async def process_queue(self):
        """消费队列，执行任务"""
        semaphore = asyncio.Semaphore(self.concurrency)
        
        while True:
            task = await self.queue.dequeue()
            if not task:
                await asyncio.sleep(1)
                continue
            
            async with semaphore:
                asyncio.create_task(self.execute_task(task))
    
    async def execute_task(self, task: GenerationTask):
        """执行单个生成任务"""
        try:
            # 1. 构建Seedance提示词
            prompt = self.build_prompt(task.request)
            
            # 2. 调用Seedance API
            result = await seedance_client.generate(
                prompt=prompt,
                duration=task.request.duration,
                resolution=task.request.resolution
            )
            
            # 3. 质量检测
            score = await self.quality_check(result.video_url)
            
            if score >= 0.8:
                task.status = "completed"
                task.result = result
            else:
                task.status = "failed"
                task.error = "Quality check failed"
            
        except Exception as e:
            task.status = "failed"
            task.error = str(e)
            # 触发重试
            await self.retry_task(task)
```

### 7.2 成本优化策略

1. **批量折扣**：与API供应商协商阶梯定价

4. **质量预筛选**：AI预筛选素材，减少废片
