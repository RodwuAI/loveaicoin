## 一、主路径：火山引擎 Doubao API 申请

### 1.1 申请流程总览

```
注册账号 ──▶ 企业认证 ──▶ 创建API密钥 ──▶ 开通模型 ──▶ 测试调用 ──▶ 正式接入
   │            │            │            │            │            │
   ▼            ▼            ▼            ▼            ▼            ▼
10分钟      1-3工作日     5分钟       即时生效     30分钟      开发集成
```

### 1.2 详细申请步骤

#### Step 1：注册火山引擎账号

**操作**：
1. 访问 https://www.volcengine.com/
2. 点击右上角「注册」
3. 填写企业邮箱（建议使用公司域名邮箱）
4. 完成手机验证
5. 设置登录密码

**所需材料**：
- 企业邮箱
- 手机号码
- 企业基本信息

**时间**：约10分钟

---

#### Step 2：企业实名认证 ⭐关键步骤

**操作**：
1. 登录火山引擎控制台：https://console.volcengine.com/
2. 进入「账号中心」→「实名认证」
3. 选择「企业认证」
4. 填写企业信息并上传材料

**所需材料**：

| 材料名称 | 格式要求 | 说明 |
|---------|---------|------|
| **营业执照** | 彩色扫描件/照片 | 需包含统一社会信用代码，清晰可见 |
| **法人身份证** | 正反面照片 | 需与营业执照法人一致 |
| **企业授权书** | 加盖公章 | 如经办人非法人，需授权书 |
| **经办人身份证** | 正反面照片 | 如适用 |
| **企业银行账户** | 开户许可证或基本存款账户信息 | 用于对公验证 |

**审核时间**：1-3个工作日

**注意事项**：
- 确保营业执照在有效期内
- 照片清晰可见，无遮挡、无反光
- 企业名称与营业执照完全一致

---

#### Step 3：创建API密钥

**操作**：
1. 进入「机器学习平台」→「API密钥管理」
2. 点击「创建API Key」
3. 填写名称（如：TimeRedeem-Production）
4. 保存生成的API Key（**仅显示一次，务必保存**）

**安全建议**：
- 生产环境和测试环境使用不同Key
- 定期轮换API Key（建议每90天）
- 不要将Key硬编码在代码中，使用环境变量

---

#### Step 4：开通视频生成模型

**操作**：
1. 进入「模型推理」→「开通管理」
2. 找到「视频生成」分类
3. 选择模型：
   - `doubao-seedance-1-5-pro`（推荐）
   - `doubao-seedance-1-0-pro`
4. 点击「开通」

**可用模型对比**：

| 模型 | 功能 | 推荐场景 |
|-----|------|---------|
| `doubao-seedance-1-5-pro` | 文生视频、图生视频 | 生产环境 |
| `doubao-seedance-1-5-pro-fast` | 快速版本 | 测试/预览 |
| `doubao-seedance-1-0-pro` | 文生视频 | 兼容性要求 |

---

#### Step 5：测试调用

**API端点**：
```
POST https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks
```

**请求示例**：
```bash
curl -X POST https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedance-1-5-pro",
    "prompt": "一位优雅的女士在花园中漫步，阳光明媚，电影级画质",
    "duration": 5,
    "resolution": "1080p"
  }'
```

**响应示例**：
```json
{
  "id": "cgt-202602150001-abc123",
  "status": "PENDING",
  "created_at": "2026-02-15T03:30:00Z"
}
```

## 二、备用路径1：可灵/Kling API 申请

### 2.1 申请流程

**操作步骤**：

1. **访问官网**：https://klingai.com/cn/dev
2. **注册账号**：使用企业邮箱注册
3. **申请权限**：进入开发者控制台，申请API权限
4. **获取密钥**：审核通过后获得：
   - AccessKey ID
   - AccessKey Secret

**所需材料**：
- 企业邮箱
- 公司基本信息
- API使用场景说明

**审核时间**：即时-1个工作日

### 2.2 API调用示例

```python
import requests

url = "https://api.klingai.com/v1/videos/generations"
headers = {
    "Authorization": f"Bearer {ACCESS_KEY_ID}:{ACCESS_KEY_SECRET}",
    "Content-Type": "application/json"
}
data = {
    "prompt": "一位优雅的女士在花园中漫步",
    "duration": 5,
    "aspect_ratio": "9:16"
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```

## 三、备用路径2：Runway Gen-3 API 申请

### 3.1 申请流程

**状态**：有限访问，需加入候补名单

**操作步骤**：

1. **访问官网**：https://docs.runwayml.com/
2. **创建账户**：使用企业邮箱注册
3. **申请API访问**：
   - 进入开发者控制台
   - 找到「API Access」
   - 点击「Join Waitlist」
   - 填写申请表单

**申请表单要点**：
- 公司/组织名称
- 使用场景描述
- 预计调用量
- 项目类型（商业/研究/个人）

**审核时间**：不确定（通常2-4周）

### 3.2 定价
