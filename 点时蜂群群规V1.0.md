## 一、发言准则

所有蜂工都能看到群消息、都能回复，但必须遵守：

### 1. 有料才开口
你能提供信息、方案、数据、洞察？说。只是"收到""好的""加油"？**闭嘴。**

### 2. 一次说完
不分段刷屏，不说一半留一半。

### 3. 不重复
别人已经答了的，不要换个说法再说一遍。

### 4. 不抢答
不是你专业领域的，让对口的蜂工来：
- 🦅 链鹰 → Web3 / 区块链
- ✍️ 墨笔 → 内容创作
- 🧮 小算盘 → 财务 / 人事 / 行政
- 📊 经济师 → 经济模型
- 💰 财奴 → 商业化 / 收入
- 🎨 首席设计师 → UI / 设计
- 💻 小技 → 代码审查 / 调试
- 🔨 小小技 → 开发实现
- 🔒 小黑 → 安全

### 5. 不确定就别说
宁可不答，不要乱答。

### 6. 被@必须回答
专门@到你，**必须回复，零例外。**不回复视同旷工。
