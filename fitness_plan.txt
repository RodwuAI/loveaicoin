🏋️ 一周健身计划

【周一】胸部 + 三头肌
- 热身：5分钟跳绳或慢跑
- 杠铃卧推：4组 x 8-12次
- 哑铃飞鸟：3组 x 12-15次
- 上斜哑铃卧推：3组 x 10-12次
- 绳索下压：3组 x 12-15次
- 仰卧臂屈伸：3组 x 10-12次
- 拉伸：5分钟

【周二】背部 + 二头肌
- 热身：5分钟划船机
- 引体向上（或高位下拉）：4组 x 8-12次
- 杠铃划船：4组 x 8-12次
- 坐姿绳索划船：3组 x 12-15次
- 哑铃弯举：3组 x 10-12次
- 锤式弯举：3组 x 12次
- 拉伸：5分钟

【周三】休息 或 轻度有氧
- 快走/慢跑 30分钟
- 或瑜伽/拉伸 20分钟

【周四】腿部 + 核心
- 热身：5分钟动感单车
- 深蹲：4组 x 8-12次
- 罗马尼亚硬拉：3组 x 10-12次
- 腿举：3组 x 12-15次
- 腿弯举：3组 x 12-15次
- 平板支撑：3组 x 30-60秒
- 卷腹：3组 x 15-20次
- 拉伸：5分钟

【周五】肩部 + 手臂
- 热身：肩部环绕 + 轻重量推举
- 哑铃推举：4组 x 8-12次
- 侧平举：3组 x 12-15次
- 前平举：3组 x 12次
- 面拉：3组 x 15次
- 杠铃弯举：3组 x 10-12次
- 三头肌下压：3组 x 12-15次
- 拉伸：5分钟

【周六】全身功能性训练 或 有氧运动
- 选项1：HIIT 20-30分钟
- 选项2：游泳/骑车/跑步 40分钟
- 选项3：CrossFit风格循环训练

【周日】完全休息
- 散步、轻度拉伸
- 为下周做准备

💡 小贴士：
- 每次训练前充分热身，训练后拉伸
- 保持水分充足
- 循序渐进增加重量
- 保证7-8小时睡眠
- 搭配均衡饮食
