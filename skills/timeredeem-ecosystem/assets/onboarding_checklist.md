# TimeRedeem Partner Onboarding Checklist

## Pre-Onboarding

### Documentation (To be sent before kickoff)
- [ ] TimeRedeem Business Overview (PPT)
- [ ] Time生态总纲 (Ecosystem Outline)
- [ ] Partnership Agreement (customized)
- [ ] Brand Guidelines & Assets
- [ ] Technical Requirements (if applicable)

### Administrative
- [ ] KYC verification completed
- [ ] Background check (for consensus nodes)
- [ ] Legal entity verification
- [ ] Tax information collected
- [ ] Payment/banking details confirmed

## Week 1: Orientation

### Day 1-2: Introduction
- [ ] Welcome call with CEO or designated partner manager
- [ ] Platform walkthrough (assets, intelligence, metaverse, philanthropy)
- [ ] Partner portal account setup
- [ ] Communication channels access (Slack/Discord)
- [ ] Assign buddy/mentor partner

### Day 3-5: Training
- [ ] Business model deep dive
- [ ] Time asset mechanics
- [ ] Consensus node operation (if applicable)
- [ ] Referral system walkthrough
- [ ] Reward calculation examples
- [ ] Governance participation guidelines

## Week 2-4: Activation

### Partner Setup
- [ ] Complete partner profile
- [ ] Set up tracking/dashboard access
- [ ] Configure notification preferences
- [ ] Join regional partner group

### First Actions
- [ ] Identify initial referral targets (minimum 3)
- [ ] Schedule first introduction calls
- [ ] Create partner introduction content
- [ ] Set monthly goals

### Technical Setup (Consensus Nodes)
- [ ] Node hardware provisioned
- [ ] Software installation completed
- [ ] Network configuration verified
- [ ] Monitoring dashboards configured
- [ ] First test block validation
- [ ] Security audit passed

## Month 2-3: Growth

### Network Building
- [ ] Achieve first 3 direct referrals
- [ ] Host first partner introduction event
- [ ] Create case study/content
- [ ] Attend monthly partner call

### Performance Tracking
- [ ] Review Week 4 performance metrics
- [ ] Adjust strategy based on data
- [ ] Identify upsell opportunities
- [ ] Request feedback on onboarding experience

## Ongoing Activities

### Monthly
- [ ] Submit activity report
- [ ] Attend partner community call
- [ ] Review reward statement
- [ ] Update referral pipeline

### Quarterly
- [ ] Business review with partner manager
- [ ] Goal setting for next quarter
- [ ] Training/updates on new features
- [ ] Peer networking event

### Annually
- [ ] Partnership agreement renewal review
- [ ] Tier upgrade assessment
- [ ] Strategic planning session
- [ ] Annual partner summit attendance

## Success Metrics

### For Time Openers
- [ ] Completed ecosystem training
- [ ] Participated in 2+ governance votes
- [ ] Referred 1+ new partner
- [ ] Active in community channels

### For Consensus Nodes
- [ ] 99.9%+ uptime maintained
- [ ] 3+ sub-nodes referred
- [ ] First reward distribution received
- [ ] Security audit passed

### For Referral Nodes
- [ ] Monthly referral target achieved
- [ ] Network depth of 3+ levels
- [ ] First commission earned
- [ ] Completed advanced training
